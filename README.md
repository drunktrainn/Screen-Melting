Screen Melter

A lightweight Windows application that creates a classic screen melting visual effect by capturing the desktop and applying a real-time graphical distortion. This project is intended for educational, demonstration, and entertainment purposes only.
This application is completely non-destructive. It only renders a temporary visual effect and does not modify your files, registry, or operating system.

Features
Classic screen melting animation
Lightweight native C++ implementation
Real-time desktop capture and distortion
Fully open source
Easy to build with Visual Studio
Well suited for learning Win32 graphics programming
Safety

This project is safe to run and does not:

Delete or modify files
Encrypt or corrupt data
Install malware or spyware
Persist after closing
Create registry entries
Disable security software
Damage hardware or Windows

Once the application exits, the effect ends immediately.

Building
Requirements
Visual Studio 2022 (or newer)
Windows SDK
C++17 or later

Simply open the solution and build in either Debug or Release mode.
